# 📄 ScanPro

Scanner foto ke PDF — **gratis, tanpa iklan, 100% offline.**

## ✨ Fitur
- 📷 Ambil foto dari kamera atau galeri
- 📄 Convert multi-foto ke 1 PDF
- 🔄 Rotate halaman
- ✨ Auto-enhance (kontras & brightness)
- 🔲 Grayscale mode
- 📏 Ukuran kertas: A4, Letter, Legal, F4
- 🔀 Drag & drop untuk reorder halaman
- 📱 PWA — bisa di-install di HP
- 🌐 Offline capable
- 🚫 Zero iklan, zero tracking

## 📱 Install di HP
1. Buka link app di browser (Chrome/Safari)
2. Klik menu ⋮ → **"Add to Home Screen"**
3. Done! Bisa dipakai offline

## 🛠️ Tech Stack
- HTML5 + CSS3 + Vanilla JS
- jsPDF untuk generate PDF
- Service Worker untuk offline support
- PWA Manifest
